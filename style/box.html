박스 모델

박스모델이란 웹 문서내용알 박스 형태로 표현하는 것을 말한다.

나열형태에 따라 블록과 인라인 이 있다

블록 : 태그 하나가 너비100%로 한 줄을 모두 차지하여 블록태그
옆에 요소가 위치할 수가 없다

인라인 : 너비가 태그의 컨테츠만큼에 해당하여
한 줄에 인라인 요소 여러 개를 위치하게 할수 있다

블록(block) div, p, form, hr, table, ul, ol, h1~h6
인라인(inline) span, strong, label, a : inline
input, textarea, button, img : inline-block
