선택자

구분 표현 설명
전체선택자 * 모든
모든선택자 태그명 해당 태그에 적용
아이디 선택자 #아이디명 해당 id가 지정된 태그에 적용
:유일해야한다
class 선택자 .class명 해당 class가 지정된 태그에 적용
:동일한 class를 여러태그에 선언가능
자식 선택자 선택자 > 자식 선택자 선택자의 바로 아래 위치하는 자식선택자에만 적용
자손 선택자 선택자 자손선택자 선택자의 아래 위치한 모든 자손선택자에만 적용(자식도 포함됨)
속성 선택자 선택자 [속성=값] 속성이 특정 값을 갖는 선택자에 적용
텍스트 크기 확인

전체선택자를 통해 html 파일내의 모든 태그에 대해 동일한 스타일을 적용할 수 있다.
